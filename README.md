# uoh-ay26-AirLLM

Exercise 05 project for **Running a Massive LLM Locally: AirLLM, Quantization and Performance Benchmarking**.

This repository is structured as a reproducible Python package for collecting hardware information, running local LLM inference experiments, saving failures as first-class results, comparing baseline/AirLLM/quantized runs, and generating a technical report.

Detailed usage and experiment discussion are expanded as the implementation progresses.
